NAME
==================================================

**alf save** - Generate aliases to `~/.bash_aliases`.

SYNOPSIS
==================================================

**alf save**

DESCRIPTION
==================================================

Generate aliases to **~/.bash_aliases**.

- Alias: **s**

ENVIRONMENT VARIABLES
==================================================

ALF_ALIASES_FILE
--------------------------------------------------

Path to bash_aliases file.

Aliases will be saved to this file.

Default: **~/.bash_aliases**
