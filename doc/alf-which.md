NAME
==================================================

**alf which** - Show the alias command.

SYNOPSIS
==================================================

**alf which** CODE [SUBCODE]

DESCRIPTION
==================================================

Show the alias command.

- Alias: **w**

ARGUMENTS
==================================================

CODE
--------------------------------------------------

Show the command for this alias code.

- *Required*

SUBCODE
--------------------------------------------------

Show the command for this alias sub-code.

==================================================
